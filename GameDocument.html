Executive Summary

Ballon Game is 3D Endless android game.

Objectives:

The main objective of the game is take as many possible Dollars and make high score each time.

Scope and Target Audience:

Any age group people can play this game to pass the time or make challenge how much.This game is not specified for specific age group.

Gameplay

When player start playing the Game the Background image in the game star moving to the left, and PLayer looks like is moving forward.The Bomb and Dollar Sign Spawn at different places and then start moving left when player(Ballon) hits the Bomb the game ends and when Player hits the DollarSign($) one point is incremented in the scores.
